# `v0.4.0` Release Notes

`v0.4.0` extends the public preview with a larger topology catalog, faster non-edge-to-edge neighbor construction, clearer simulation-speed feedback, and a broader repo tooling surface for contributors.

## Highlights

- Expanded the shipped catalog to 46 tiling families: 3 regular grids, 27 periodic mixed tilings, and 16 aperiodic patch families.
- Added `pinwheel-2-1`, a two-prototile aperiodic pinwheel-family substitution.
- Added multiple periodic mixed tilings, including Pythagorean, Herringbone, Basketweave, Stein 14 Pentagonal, Pentagon Crosses, and two 2-uniform triangle/hex/square variants.
- Reworked simulation speed controls so the UI distinguishes target speed from measured actual cadence.
- Sped up segment-overlap neighbor detection for non-edge-to-edge aperiodic tilings and periodic-face neighbor attachment for larger mixed tilings.
- Consolidated repo tooling under the Python `tools` CLI and added onboarding/examples for common contributor workflows.

## User-Facing Changes

- New topology: `pinwheel-2-1`
- New topology: `pythagorean`
- New topology: `herringbone`
- New topology: `basketweave`
- New topology: `triangular-square-2uniform`
- New topology: `trihex-2uniform-3636-3366`
- New topology: `stein-14-pentagonal`
- New topology: `pentagon-crosses`
- Speed control now shows the selected target and the measured actual rate while the simulation is running.
- Deeper preview patches for segment-overlap tilings load much faster, especially `pinwheel-2-1`, `chair`, `penrose-p2-kite-dart`, `robinson-triangles`, and `tuebingen-triangle`.

## Topology Notes

- `pinwheel-2-1` uses an exact-arithmetic similarity-substitution path with small/large triangle prototile kinds and segment-overlap neighbor detection.
- `pinwheel` remains labeled `Experimental`; the exact substitution path is repaired, but promotion still depends on manual visible review.
- `dodecagonal-square-triangle` now ships as a decorated periodic `3.12.12` square-triangle generator that tiles the plane exactly and scales without vendored data. It remains documented as periodic, not the canonical Schlottmann quasi-periodic marked-prototile substitution.
- `Stein 14 Pentagonal` and `Pentagon Crosses` expand the convex pentagonal periodic catalog, alongside Cairo, Prismatic, Floret, Type 7, and Snub Square Dual.

## Tooling And Quality Improvements

- Added and hardened sketch/descriptor tooling for periodic-face topology development.
- Centralized repo commands under `python -m tools ...`, with npm wrappers using the same internal path.
- Added onboarding and examples for running the app, inspecting topologies, rendering patches, simulating steps, and verifying against specs.
- Strengthened performance-sensitive neighbor construction with regression coverage for exact and float segment-overlap line bucketing.
- Refreshed topology fixtures, source references, and verification metadata for the expanded catalog.

## Known Limitations

- This preview line is still a source release and GitHub Pages demo, not an npm or PyPI package.
- The standalone demo still depends on CDN-hosted Pyodide assets and is not an offline bundle.
- Some aperiodic families still ship as `canonical_patch`, exact-affine representative patches, or otherwise documented approximations rather than full symbolic or literature-canonical substitution implementations.
- `pinwheel` remains experimental until manual visual review justifies promotion.
