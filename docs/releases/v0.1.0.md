# `v0.1.0` Release Notes Draft

`v0.1.0` is the first public preview release of Cellular Automaton Lab. It is aimed at people who want to explore topology-first cellular automata, evaluate the current architecture, or contribute to the project while the public surface is still intentionally narrow.

## Highlights

- Browser-based cellular automata playground with one shared rule protocol across regular, mixed periodic, and aperiodic boards.
- Fifteen shipped tiling families spanning square, hex, triangle, Archimedean, Cairo, Penrose, and Ammann-Beenker boards.
- Sixteen built-in rules covering Life-like, mixed-tiling Life, excitable, and signal-system behavior.
- Public standalone demo published to GitHub Pages from the same frontend/runtime surface exercised by CI.
- Layered validation stack with frontend typing and tests, Python typing and tests, tiling verification, standalone smoke coverage, and Playwright coverage for both server and standalone hosts.

## Who This Is For

This preview is for evaluators, contributors, and technically comfortable users who want a public demo plus a source-available project they can run locally. It is not yet framed as a production-ready packaged library or a stable integration target.

## Known Limitations

- `pinwheel` remains `Experimental` pending stronger manual visible review.
- `dodecagonal-square-triangle` currently ships as a structurally exact periodic square-triangle generator, not the canonical Schlottmann quasi-periodic marked-prototile construction.
- The standalone demo still depends on CDN-hosted Pyodide assets and is not an offline bundle.
